Chapter 3: Templates and Static Files

Using Templates

Flask allows you to use templates to create dynamic HTML pages. Templates are HTML files with placeholders for dynamic data.

Rendering Templates

To render a template, you need to use the `render_template()` function and pass the template name along with any necessary data.

from flask import Flask, render_template

app = Flask(__name__)

@app.route('/profile/')
def profile(username):
    return render_template('profile.html', username=username)

if __name__ == '__main__':
    app.run()

Serving Static Files

Flask can serve static files such as CSS stylesheets, JavaScript files, and images. You can store them in a directory named `static`.

<link rel="stylesheet" href="{{ url_for('static', filename='styles.css') }}">
