Chapter 7: Error Handling and Logging

Error Handling

Flask provides mechanisms to handle different types of errors that can occur during the execution of your application.

Custom Error Pages

You can create custom error pages to display specific messages or content when an error occurs.

from flask import Flask, render_template

app = Flask(__name__)

@app.errorhandler(404)
def page_not_found(error):
    return render_template('404.html'), 404

if __name__ == '__main__':
    app.run()

Logging

Logging allows you to record events, errors, and important information during the execution of your Flask application.

import logging
from flask import Flask

app = Flask(__name__)
app.logger.setLevel(logging.INFO)

@app.route('/')
def index():
    app.logger.info('Index page visited')
    return 'Hello, World!'

if __name__ == '__main__':
    app.run()
